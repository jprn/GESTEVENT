Questions fréquentes

Comment fonctionne la version gratuite ?

La version gratuite vous permet de créer un événement actif avec jusqu'à 20 participants. Vous bénéficiez de toutes les fonctionnalités de base comme la page d'événement, les inscriptions et les QR codes pour le check-in. C'est parfait pour tester la plateforme ou pour de petits événements occasionnels.

Comment sont gérés les paiements ?

Nous utilisons Stripe pour traiter tous les paiements de manière sécurisée. Les fonds sont directement versés sur votre compte Stripe, moins notre commission (1% pour le plan Pro, 2% pour l'Event Pack). Vous pouvez ensuite transférer l'argent sur votre compte bancaire selon les délais de Stripe.

Puis-je personnaliser l'apparence de ma page d'événement ?

Absolument ! Notre éditeur de page vous permet de personnaliser les couleurs, les polices, d'ajouter des sections comme un programme détaillé, une galerie d'images, des témoignages et bien plus encore. Avec le plan Pro, vous avez accès à encore plus d'options de personnalisation.

Comment fonctionne le check-in le jour de l'événement ?

Le jour de votre événement, vous pouvez accéder à l'interface de check-in depuis votre smartphone ou tablette. Il vous suffit de scanner les QR codes présents sur les billets de vos participants pour valider leur présence en temps réel. Vous pouvez aussi effectuer un check-in manuel si nécessaire.

Est-il possible d'obtenir des statistiques sur mon événement ?

Oui, tous les plans incluent des statistiques de base comme le nombre d'inscrits, le taux de remplissage et l'historique des check-ins. Le plan Pro offre des statistiques plus avancées, comme l'évolution des inscriptions dans le temps, le taux de conversion des visiteurs et des rapports détaillés après l'événement.
